# Gorka Ibeas - Vehículos de Ocasión

Sitio web profesional para la venta de vehículos de ocasión.

## Estructura del Proyecto

- `index.html`: Página de inicio.
- `nosotros.html`: Información sobre la empresa.
- `servicios.html`: Listado de servicios.
- `galeria.html`: Galería de imágenes.
- `contacto.html`: Información de contacto.
- `style.css`: Hoja de estilos principal.
- `script.js`: Scripts del sitio.
